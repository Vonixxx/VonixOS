# VonixOS | Personal System Configuration using NixOS
This configuration was designed with minimalism in mind.

## Contents
- **WM**       - Sway
- **Bar**      - Waybar
- **Editor**   - Helix
- **Terminal** - Kitty
- **Launcher** - Fuzzel
- **Browser**  - Firefox (ArkenFox)
